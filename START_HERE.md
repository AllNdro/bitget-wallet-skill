## 💡 HOW THIS WORKS

### What You're Building
An AI-powered DeFi agent that:
1. Listens to natural language commands
2. Parses intent using Claude 3.5 Sonnet
3. Checks your wallet balance
4. Executes DeFi operations on Mantle
5. Reports results back to you

### Example Command
```
You: "Stake 100 MNT"
Agent:
  1. ✓ Checks balance (you have 100 MNT)
  2. ✓ Gets current APY (5.2%)
  3. ✓ Estimates gas (0.003 MNT)
  4. Asks: "Proceed with stake?"
  5. Executes on Mantle
  6. Reports: "Staked! Earning 5.2% APY"
```

### Why Mantle?
- 99.9% cheaper gas than Ethereum
- 80x smaller minimum portfolio size
- Fast transactions (2 seconds)
- You still control your keys (non-custodial)
